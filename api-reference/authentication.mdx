# 🔐 Authentication

All endpoints require API Key.

## 🔑 How to Get Your Key
1. Create an account at [app.apeninja.co](https://app.apeninja.co)
2. Go to “Developer Settings”
3. Generate and copy your API key

### Headers Format

```bash
Authorization: Bearer <YOUR_API_KEY>```

## Rate Limits
60 requests/minute by default

Custom limits available for partners
