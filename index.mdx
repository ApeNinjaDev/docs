# ApeNinja API

Welcome to the ApeNinja API reference. Use our API to programmatically execute trades, manage your portfolio, and handle authentication across multiple blockchain networks.

<Card title="Base URL" icon="server" iconType="duotone">
  All API requests should be made to:
  ```bash
  https://api.apeninja.co
  ```
</Card>

## Authentication Methods

<CardGroup cols={2}>
  <Card title="Bearer Token" icon="key" iconType="duotone">
    After login, you'll receive a JWT token to include in the Authorization header:
    ```
    Authorization: Bearer YOUR_JWT_TOKEN
    ```
  </Card>
  <Card title="API Key" icon="lock" iconType="duotone">
    Alternatively, use the API key from your registration email:
    ```
    x-api-key: YOUR_API_KEY
    ```
  </Card>
</CardGroup>
