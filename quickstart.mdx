## Get Started in Minutes

Follow these simple steps to start trading like a ninja:

<Steps>
  <Step title="Sign Up">
    Visit [app.apeninja.co](https://app.apeninja.co/) and sign up with your email or connect your wallet.
    
    <Info>
      **No wallet? No problem!** ApeNinja allows you to create an account with just your email address. We'll handle setting up a secure wallet for you behind the scenes.
    </Info>
  </Step>
  
  <Step title="Get Your API Key">
    After signing up, navigate to the **Settings** section to find your personal API key. This key will allow you to access ApeNinja from various platforms.
    
    <Warning>
      Keep your API key secure and never share it with others. This key provides access to your ApeNinja account.
    </Warning>
  </Step>
  
  <Step title="Deploy Your SuperAccount">
    Deploy your SuperAccount smart contract wallet on your chosen blockchain networks. This is a one-time setup per chain that enables lightning-fast trades.
    
    ```bash
    # You'll see a transaction request to deploy your SuperAccount
    # This is a one-time gas fee per chain
    ```
    
    <Note>
      SuperAccount deployment is a one-time action per chain. Once deployed, you can execute trades in a single transaction without separate approval steps.
    </Note>
  </Step>
  
  <Step title="Fund Your Account">
    Transfer assets to your SuperAccount from your existing wallet or purchase directly through our fiat on-ramp partners.
  </Step>
  
  <Step title="Start Trading">
    You're all set! Start trading from any of our supported platforms:
    
    - [Web App](https://app.apeninja.co)
    - [Telegram Bot](https://t.me/apeninjabot)
    - [Twitter Bot](https://twitter.com/apeninjabot)
    - [Chrome Extension](https://chrome.google.com/webstore/detail/apeninja)
  </Step>
</Steps>

## Sample Commands

Here are some basic commands to get you started with our Telegram bot:

<CodeGroup>
  ```bash Telegram
  /buy ETH 0.1 USDT    # Buy 0.1 ETH with USDT
  /sell ETH 50%        # Sell 50% of your ETH holdings
  /balance             # Check your wallet balance
  /price ETH           # Get current ETH price
  /chart ETH           # Get ETH price chart
  ```
  
  ```javascript API
  // Sample API call to execute a trade
  const response = await fetch('https://api.apeninja.co/v1/trade', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer YOUR_API_KEY'
    },
    body: JSON.stringify({
      chainId: 1,          // Ethereum mainnet
      tokenIn: 'USDT',     // Token to sell
      tokenOut: 'ETH',     // Token to buy
      amountIn: '100',     // Amount to sell (100 USDT)
      slippage: 0.5        // 0.5% slippage tolerance
    })
  });
  
  const data = await response.json();
  console.log(data);
  ```
</CodeGroup>
